금방 작성해서 끝내려고 했는데, 집안 일이 좀 있었고, 그림 형태를 생각하고 그리는데 시간을 너무 많이 쏟았다. 유니코드 박스 드로잉 그냥 마우스 쓱쓱 하면 자동으로 해주는 그런 프로그램 없나? 매번 그릴 때마다 너무 짜증이 난다.

이 일을 시작하고 항상 시간에 쫒긴다. 그러다보니 잠을 줄이게 되고, 잠이 부족하니 생각이 좀 느리게 흘러가는 면이 없지 않다. 그러니까, 효율이 떨어지는 것이다.
